# 추가 근거 보강 TODO

현재 공개 문서 초안은 기존 phase 기록에서 확인 가능한 수치와 의사결정 근거를 기반으로 작성했다. 최종 제출 전 아래 항목은 필요하면 보강한다.

| 항목 | 상태 | 보강 방법 |
| --- | --- | --- |
| 사용자 인터뷰 또는 설문 근거 | 미보유 | 발표 자료에 정성적 문제 정의로만 사용 |
| 외부 항공권 가격 변동 연구 인용 | 선택 사항 | 필요 시 공개 출처 조사 후 추가 |
| 최종 모델의 blind holdout 해석 | 제한 있음 | Phase 16B-3C의 해석 한계를 문서에 유지 |
| route별 상세 성능표 | 일부 내부 문서에 존재 | 공개 문서에는 요약만 반영하거나 민감 정보 제거 후 별도 표 작성 |
| APK 배포 방식 | 별도 단계 | 배포 링크 확정 후 README에 반영 |
| 공개용 아키텍처 이미지 | 미작성 | 필요 시 민감 정보 없는 다이어그램 작성 |

## 주의

근거가 부족한 항목은 과장하지 않는다. 특히 추천 결과는 가격을 보장하는 기능이 아니라 관측 이력과 모델 출력 기반의 구매 보조 정보로 설명한다.
