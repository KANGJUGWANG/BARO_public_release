# Frontend Guide

## 실행

```bash
cd frontend
npm install
npm run build
```

개발 서버:

```bash
npm run dev
```

## 주요 화면

- 로그인 화면
- 홈 검색 화면
- 편도/왕복 검색 결과 카드
- 상세 페이지
- 모델 안내 화면
- 노선 분석 화면
- 저장 목록 / 설정 / Drawer 메뉴

## UI 표시 정책

- BUY / WAIT 추천 결과 표시
- 가격 하락 기대 강도 게이지 표시
- 상세 페이지에서는 가격 추이와 모델 판단 정보를 함께 표시

## Android

Capacitor Android 프로젝트 구조가 포함되어 있습니다. 실제 release signing key는 공개 레포에 포함하지 않습니다.
