# BARO 재현 가능 범위 초안

## 공개 저장소로 재현 가능한 것

- frontend 화면 구조와 build 흐름
- backend API의 공개용 구조
- BUY/WAIT decision rule의 개념
- Stage1/Stage2 모델 구조 설명
- 편도/왕복 분리 contract
- 최종 데이터 범위와 주요 지표 요약
- 환경 변수 placeholder 기반 로컬 설정 방식

## 공개 저장소만으로 재현할 수 없는 것

- 동일한 운영 모델 추론 결과
- 동일한 학습 데이터셋 생성
- 동일한 운영 DB 관측 이력
- 동일한 모델 바이너리 로딩
- 실제 외부 API 키 기반 운영 수집
- 실제 운영 서버 배포 상태

## 별도 보관이 필요한 것

- 모델 바이너리와 metadata 원본
- 원천 DB 또는 학습 materialization output
- 운영 환경 설정값
- 실험별 raw output
- 발표용 APK 산출물

## 재현성 설명 문구

이 저장소는 BARO의 서비스 구조, 데이터 흐름, 모델 적용 방식, UI 구현을 검토할 수 있게 공개한다. 다만 실제 운영 모델과 원천 데이터는 용량과 보안 문제로 포함하지 않으므로, 공개 저장소만으로 최종 운영 예측값을 완전히 재현하는 것은 범위 밖이다.
