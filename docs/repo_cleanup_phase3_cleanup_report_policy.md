# Phase 3 Cleanup Report Policy

Detailed cleanup CSV files and scan manifests are not included in the public release repository because they can create unnecessary security noise and expose internal review structure.

The public repository keeps only concise summary documents:

- `docs/public_release_audit_summary.md`
- `docs/repo_cleanup_phase3_pre_push_summary.md`
- focused Phase 3 final gate summaries

Detailed internal cleanup reports remain outside the public repository.
